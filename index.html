Notice d'utilisation

FlowForge est une application de productivité orientée apprentissage, combinant Pomodoro, tâches récurrentes, objectifs, traduction, dictionnaire personnel et suivi de progression.

🍅 1. Contextes Pomodoro

  • + Nouveau : Ajoutez un contexte depuis l'en-tête du bloc Contextes.
  • Contrôles : Start/Pause/Reset pour gérer vos sessions de 25 minutes.
  • Cycles automatiques : Alternance travail → pause courte (5 min) → pause longue (15 min après 4 cycles).
  • Stigmergie : Zone de texte pour noter la prochaine action et faciliter la reprise.
  • Liaison tâches : Si une tâche porte le même nom qu'un contexte, la progression se synchronise automatiquement.

📊 2. Insights (Stats / Régularité / Objectifs)

  • Onglet Stats : Synthèse du temps de focus, contextes actifs, objectifs complétés (jour/semaine/mois/année).
  • Score hebdomadaire : Calcul combiné sur focus, tâches et vocabulaire.
  • Graphiques : Visualisation de la progression sur 7 jours, 30 jours, 12 mois et l'année complète.
  • Onglet Régularité : Graphique de vos 7 derniers jours de productivité avec détails par jour.
  • Onglet Objectifs : Vue d'ensemble des objectifs journaliers, hebdomadaires, mensuels et annuels (automatiques + manuels).
  • 🎪 Grâce (Joker) : Bouton pour valider automatiquement votre journée (1 fois/semaine, ajoute le temps manquant pour atteindre l'objectif du jour).
  • Réinitialisation : Bouton pour remettre les statistiques à zéro (conserve tâches, objectifs et dictionnaire).

✔️ 3. Tâches à faire & Objectifs

  • Création de tâche : Nom, périodicité (quotidienne/hebdo/mensuelle/annuelle/unique), temps dédié par jour.
  • Synchronisation contexte : Une tâche nommée identiquement à un contexte se lie automatiquement (insensible à la casse).
  • Badges intelligents : Affichage dynamique de la progression (Jour X/Y) avec code couleur (rouge/orange/vert).
  • Validation manuelle : Cochez une tâche pour la marquer comme complétée (compte dans les objectifs même sans temps de focus).
  • Bouton Reset (↺) : Réinitialise la progression d'une tâche récurrente (repart à Jour 1).
  • Badge "LIE AU CONTEXTE" : Confirme qu'une liaison active existe entre tâche et contexte.
  • Aide compacte : Le panneau "Comment ça marche ?" est repliable pour gagner de la place.
  • Objectifs automatiques : Les tâches génèrent automatiquement des objectifs selon leur périodicité.
  • Objectifs manuels : Créez des objectifs personnalisés (journaliers/hebdo/mensuels/annuels) indépendants des tâches.

📝 4. Notes, Journal, Traducteur, Dictionnaire

  • Notes : Prise de notes libre avec sauvegarde automatique et export texte (.txt).
  • Journal : Ajout d'entrées horodatées (Ctrl+Entrée pour valider) avec export dédié et recherche.
  • Traducteur bidirectionnel : Anglais → Français ou Français → Anglais.
  • Choix d'API : Sélection de l'API préférée (SimplyTranslate, Lingva, MyMemory, LibreTranslate) avec fallback automatique.
  • Dictionnaire - Sauvegarde : Les traductions sont automatiquement enregistrées avec système de révision espacée.
  • Dictionnaire - États : 🔴 Rouge (nouveau) → 🟠 Orange (en révision) → 🟢 Vert (maîtrisé).
  • Dictionnaire - Révision : Mode pratique FR → EN ou EN → FR (révélation au clic).
  • Dictionnaire - Recherche : Barre de recherche en temps réel pour filtrer vos mots.
  • Dictionnaire - Import CSV : Importez un fichier CSV (format: mot,traduction,état) pour ajouter des mots en masse.
  • Dictionnaire - Export CSV : Téléchargez votre dictionnaire complet au format CSV.
  • Exemples Linguee : Bouton dédié sur chaque mot du dictionnaire pour ouvrir des exemples de contexte.
  • Dictionnaire - Synchronisation : Les traductions du dictionnaire apparaissent automatiquement quand vous traduisez un mot déjà connu.
  • Objectif vocabulaire : Cible hebdomadaire de mots à passer en vert (compte dans le score hebdomadaire à 25%).

⚙️ 5. Menu Actions (accès centralisé)

  • 📖 Notice : Affiche cette aide complète.
  • ⚡ Vitesse : Accélère le temps (×1 à ×240) pour tester rapidement vos workflows.
  • 🎪 Grâce - Activer : Active/désactive le système de joker hebdomadaire (valide automatiquement 1 journée par semaine).
  • 📥 Exporter : Télécharge une sauvegarde JSON complète de toutes vos données.
  • 📤 Importer : Restaure vos données depuis une sauvegarde JSON (⚠️ écrase les données actuelles).
  • 🎯 Mode Focus : Masque tous les panneaux sauf les contextes pour concentration maximale.
  • 🧩 Disposition des blocs : Réglage des largeurs dans une section repliable (Contextes, Insights, Tâches, Notes).
  • 📊 Historique : Affiche les 30 dernières sessions Pomodoro (contexte, durée, horodatage).

🎨 6. Personnalisation interface

  • Drag & Drop des grands blocs : Déplacez les panneaux via les poignées ⋮⋮ (ordre mémorisé).
  • Hauteurs alignées : Les 4 grands blocs restent alignés pour une lecture propre.
  • Textareas redimensionnables : Glissez le coin inférieur droit pour agrandir Notes/Traducteur.
  • Sauvegarde automatique : Toutes vos données et préférences UI sont conservées dans le navigateur.

⌨️ 7. Raccourcis clavier

  • Ctrl+Shift+E : Exporter une sauvegarde
  • Ctrl+Shift+I : Importer une sauvegarde
  • Ctrl+Shift+H : Afficher l'historique des sessions
  • Ctrl+Shift+F : Activer/désactiver le mode focus
  • Ctrl+Entrée : Valider entrée journal / traduire texte / sauvegarder un snippet
  • Escape : Fermer les fenêtres modales ouvertes

📱 8. Installation (Progressive Web App)

  • Mobile : Menu navigateur → "Ajouter à l'écran d'accueil" (Chrome/Safari).
  • Desktop : Icône d'installation dans la barre d'adresse (Chrome/Edge).
  • Hors-ligne : L'application fonctionne sans connexion après installation (Service Worker).
  • Mise à jour : Rechargez (Ctrl+F5) en cas de comportement anormal après mise à jour.

💡 Conseils Pro :
• Utilisez la vitesse ×120 ou ×240 pour simuler plusieurs jours de travail et visualiser la progression de vos tâches récurrentes.
• La 🎪 Grâce se réinitialise chaque lundi : utilisez-la stratégiquement les jours difficiles !
• Créez des tâches avec le même nom que vos contextes Pomodoro pour un suivi automatique du temps.
• Utilisez le mode pratique du dictionnaire + Linguee pour mémoriser plus vite en contexte réel.
